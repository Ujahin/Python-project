# Retail Sales Analytics: Descriptive, Predictive & Prescriptive Analysis

## Project Overview
This project performs a comprehensive three-tier analysis on a retail sales dataset 
containing records of items sold to customers across multiple stores over several years. 
The analysis covers descriptive, predictive, and prescriptive approaches to uncover 
business insights and actionable recommendations.

## Objectives
- Perform descriptive, predictive, and prescriptive analysis on assigned items and store
- Compare assigned items against each other and the overall market
- Identify patterns in customer base, transaction methods, and store performance
- Provide data-driven business recommendations

## Dataset
- **Source:** `case-study-data.xlsx`
- **Contents:** Sales transactions across multiple items, stores, and customers over several years
- **Note:** Data cleaning applied on the `unit` column in `Fact_table` and `Item_dim` sheets

## Analysis Structure

### 1. Descriptive Analysis
- Sales trends over time for assigned items and store
- Customer segmentation and purchase behavior
- Transaction method distribution
- Market share comparison

### 2. Predictive Analysis
- Sales forecasting for assigned items
- Viability assessment for new store opening
- Customer discount impact prediction
- Payment method discount analysis

### 3. Prescriptive Analysis
- Recommendations on discount strategies by customer group
- Payment method optimization suggestions
- Store performance improvement recommendations
- Market expansion insights

## Tools & Technologies
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
- **Environment:** Google Colab / Jupyter Notebook

## How to Run
1. Clone this repository
2. Upload `case-study-data.xlsx` to your working directory
3. Open the notebook and run all cells sequentially

## Results
All analysis, visualizations, and recommendations are contained in a single notebook.
